# 마크다운 문법

# 1. 마크다운이란?

> 마크업 언어의 일종으로 존 그루버와 애런 슈워츠가 만들었다.
> 택스트 문서의 양식을 편집하는 문법이다.
> README 파일이나 온라인 문서, 혹은 일반 텍스트 편집기로 문서 양식을 편집할 때 사용된다.
> 그루버는 마크다운으로 작성된 문서를 HTML로 변환하는 Perl 스크립트도 만들었다.

# 2. 문법

## 2.1 문단 제목

html에서 h1~h6까지 있는것 처럼 마크다운 언어에서는 #을 사용하며 6단계 까지 있다.

```
# h1
## h2
### h3
#### h4
##### h5
###### h6
```

## 2.2 문단 목록

```
* 안이 까만색으로 채워져 있는 동그라미
    * 안이 비어있는 동그라미
- 이렇게 써도
- 됩니다.

1. 첫째
    2.둘째
```

## 2.3 글자 스타일

```
**굵게** 쓰거나 __두껍게__ 쓰거나
*기울게* 쓰면서 _기울게_ 쓴다
~~지우기~~
```
